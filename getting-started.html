Introduction

In accordance with Rio practices, RioSpan is built as a multi-module Maven project. As Rio is built on Jini, some understanding of Jini's approach to distributed services can be helpful in understanding the relationships between the Maven modules.

Extending RioSpan

Rio is configured with a configuration file called an opstring. Infinispan also has its own configuration file structure. RioSpan essentially provides the ability to declare a Rio opstring that deploys an Infinispan cluster of members to a Rio Cybernode cluster.
